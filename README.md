# yoshi-site

## マトリックス風カメラ

Webカメラ（スマホのインカメ）の映像をリアルタイムで「マトリックス風」のエフェクトに変換するアプリケーションです。

### 機能

- Webカメラの映像をリアルタイムで取得
- 映像の明度に応じて半角カタカナや数字に変換
- ネオングリーンの文字でマトリックス風に描画
- 黒背景とグロー効果でサイバーな雰囲気を演出

### 使い方

1. `index.html`をブラウザで開く
2. 「カメラを起動」ボタンをクリック
3. カメラへのアクセス許可を求められたら「許可」を選択
4. マトリックス風に変換された映像が表示されます
5. 「停止」ボタンでカメラを停止できます

### 技術スタック

- HTML5 Canvas
- Web API (`navigator.mediaDevices.getUserMedia`)
- JavaScript (ES6+)
- CSS3

### ファイル構成

- `index.html` - メインのHTMLファイル
- `style.css` - スタイリング（黒背景、ネオングリーンテーマ）
- `script.js` - カメラ取得とマトリックス風エフェクトの実装

### 動作環境

- モダンブラウザ（Chrome、Firefox、Edge、Safariなど）
- Webカメラまたはスマートフォンのインカメラ
- HTTPS接続またはlocalhost（カメラアクセスにはHTTPSが必要な場合があります）

### 特徴

- リアルタイム処理（`requestAnimationFrame`を使用）
- レスポンシブデザイン（画面サイズに自動調整）
- 明度に応じた動的な文字選択と透明度調整
